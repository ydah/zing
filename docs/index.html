Install

Get zing in minutes.

Homebrew

brew tap ydah/zing
brew install zing

Build from source (Zig 0.15.x)

git clone https://github.com/ydah/zing.git
cd zing
zig build -Doptimize=ReleaseFast
sudo cp zig-out/bin/zing /usr/local/bin/

Shell integration

# bash
source <(zing init bash)

# zsh
eval "$(zing init zsh)"

# fish
zing init fish | source
